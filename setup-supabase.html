🚀 Supabase Setup Guide

Step 1: Install Supabase CLI

Open your terminal and run:

npm install -g supabase

Or if you have Docker Desktop installed, you can skip to Step 3.

Step 2: Initialize Project

In your project folder, run:

supabase init

This creates a supabase folder with configuration files.

Step 3: Start Local Server

Run the local Supabase stack:

supabase start

This will start PostgreSQL, Auth, Storage, and the Dashboard locally.

⚠️ Requires Docker Desktop to be running!

Once started, you'll see URLs like:

  • API URL: http://localhost:54321
  • Dashboard: http://localhost:54323
  • Anon Key: (shown in terminal)

Step 4: Apply Database Schema

Copy the contents of schema.sql and run it in the SQL Editor:

  1. Go to http://localhost:54323
  2. Navigate to SQL Editor
  3. Paste the schema and click Run

Or run via CLI:

supabase db reset

Step 5: Update App Configuration

Create a file called config.js in your project root:

const SUPABASE_URL = 'http://localhost:54321';
const SUPABASE_ANON_KEY = 'your-anon-key-from-terminal';

const supabase = window.supabase.createClient(SUPABASE_URL, SUPABASE_ANON_KEY);

Then include the Supabase client in your HTML before your app script:

<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>
